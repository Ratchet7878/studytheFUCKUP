# Modify the time as you need!

Using the tkinter library, we block you from fucking around while you should be studying! 
It is the standard GUI (Graphical User Interface) framework built directly into Python 3.14, meaning you do not need to install any new libraries to make it work.
When you click "I will get back to work," that physical mouse click registers with pynput in the background. If we didn't explicitly set is_active = False immediately after the window closes, your next 5-minute cycle would start with 1 second already on the clock.
